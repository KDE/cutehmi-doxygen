uint cutehmi::xml::portFromString ( const QString &  string,
bool &  ok 
)

Convert string to a port number.

Extends standard QString::toUint() function by the ability of picking random port number in a deterministic fashion. If string contains pattern 'randPlus([0-9]*)', then randomly selected port number will be used. Number '[0-9]*' indicates a number that should be added to the randomly selected port.

Parameters
string string to convert.
ok set to true when conversion was succesful, otherwise set to false.
Returns
port number.

CUTEHMI_XML_API uint cutehmi::xml::randomPortBegin

Minimal port number of a random port.

By default a number defined by IANA for a dynamic port range is used (http://www.rfc-editor.org/rfc/rfc6335.txt).
