bool cutehmi::xml::ParseHelper::skipToNextSiblingElement ( )
protected

Skip to next sibling element.

This function sets reader on the next start tag of next sibling element. It skips all children elements (if any). If there are no more sibling elements function returns false.

Returns
true, when next sibling element has been reached, false otherwise.
