Detailed Description

template<typename T, std::size_t N = 65536>
class cutehmi::modbus::internal::DataContainer< T, N >::KeysIterator

Keys iterator.

This is read-only, thread-safe version of keys container iterator. It shares lock with data container, so that it won't enter iterating function while items are being inserted or removed and vice versa.
