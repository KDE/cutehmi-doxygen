Detailed Description

Project plugin loader.

Objects of this class can not be instantiated directly. Instance of ProjectPluginLoader is passed to the class implementing IProjectBackend interface.

Add plugin node to the given parent node.

Beside creating a node, this function will load a plugin and initialize it. Plugin object is accessible through an extension mechanism of newly created ProjectNode (call cutehmi::ProjectNode::extension(cutehmi::Plugin::staticMetaObject.className()) to access Plugin object).

Parameters
namename of the plugin. Suffix 'd' will be appended to the binary name in the debug mode.
reqMinorminimal required minor version of the plugin.
parentNodenode to which plugin node should be added.
Returns
newly created plugin node. Exception will be thrown in case of failure.
