Detailed Description

Data block.

Data block is meant to act as a container, which serves its purpose through inheritance. It provides interface, which gives derived classes the ability to be chained together. Each data block object can have a data block child appended to it. This functionality is similar to singly linked list, however the objective is to make data block extensible, rather than manipulative. Instances of this class are non-copyable, but movable.

Principle:
"cutehmi::DataBlock-determined_destruction_order"
Child objects are destroyed in the reverse order as they were appended.

void cutehmi::DataBlock::append ( std::unique_ptr< DataBlock >  block )

Append data block.

Data block is appended to the last child in the chain.

Parameters
block data block to be appended.
Note
This class is not intended to be used with long data chains. The amount of time used by append operation is linearly proportional to the size of the chain; O(n) in terms of algorithmic complexity.
